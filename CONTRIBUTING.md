        # Contributing Guide

Thanks for your interest in contributing! Here's how to get started:

## Setup

1. Fork the repo
2. Clone your fork
3. Run `npm install` (or relevant setup)

## Branch Naming

Use this format:
`feature/your-feature-name` or `bugfix/your-bug-name`

## Making Changes

-   Make atomic commits
-   Write clear commit messages
-   Keep PRs small and focused

## Pull Request Checklist

-   [ ] Code is linted
-   [ ] Tests pass
-   [ ] Descriptive title and summary
