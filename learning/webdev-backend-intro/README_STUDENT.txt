Follow these steps to try the app:
1. Open MAMP, start servers
2. Open browser to http://localhost/webdev-backend-intro/index.html
3. Fill out the form and click Submit
4. View your message on http://localhost/webdev-backend-intro/view.php
5. Try editing or deleting entries
6. Look inside phpMyAdmin to see the saved data in SQL

Explore how back-end code stores and retrieves data!
